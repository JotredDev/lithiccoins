This is a tool for automatically creating the textures for minted coins. To use it, follow these instructions:

1. Create a depth mask for your design with the following properties:
    a. 16x16 image
    b. 3 colors needed: #202020 (dark shade), #404040 (medium shade), #606060 (light shade)
    c. In the same folder as this README is

2. Open auto_mint.py
3. Go to the bottom, replace the example depth_mask_name with the name of your own depth mask file
4. Run auto_mint.py
5. Take the resulting coin textures from /results/
